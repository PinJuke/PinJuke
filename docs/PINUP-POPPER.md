# Pinup Popper

PinJuke is prepared to get integrated in *Pinup Popper*. To achieve this, you need to create a new *Emulator* entry and point the *Games Folder* to `Configs\Playlists` (e.g. `C:\vPinball\PinJuke\Configs\Playlists`).
